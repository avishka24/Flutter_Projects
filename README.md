## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/Flutter_Projects.git
```
  ##### Move to the desired folder
```bash
cd \[desired_folder]
```

  ##### To run the app, simply write
```bash
flutter run
```
